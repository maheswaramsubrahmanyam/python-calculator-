Functions can be broadly divided into two categories:\n\n- **Single Row Functions (Scalar Functions):** Work on individual rows and return one result per row.\n- **Group Functions (Aggregate Functions):** Work on multiple rows and return one single result for the entire set.\n\n---\n\n## 1. employee roles into lowercase letters.\n\n---\n\n#### ðŸ”¹ INITCAP()\nMakes the first letter of each word uppercase and the rest
